# Notes Folder Guidance

The root `notes/` folder is for author-maintained notes and lightweight project guidance.

Agents should only create or edit these files at the root of `notes/`:
- `README.md`
- `AGENTS.md`

If an agent creates a new note, plan, research writeup, simulator design artifact, or other generated working document, it should go under `notes/agent_generated/` instead of the root.

Suggested structure:
- `notes/agent_generated/design/` for agent-authored design and planning notes
- `notes/agent_generated/research/` for agent-authored research summaries and reading maps

Agents should not move, rewrite, or reorganize author-owned root notes unless explicitly asked.
